## ১. সিঙ্ক কীভাবে চলে

SheetSync পুরো শিট একবারে ইমপোর্ট করে না (টাইমআউট হতে পারে)। তিনটি স্তর:

| স্তর | কী করে | ক্রন ছাড়া কাজ করে? |
|------|--------|---------------------|
| **Sync Now** | ব্রাউজারে ব্যাচ চালায় | ✅ হ্যাঁ — ট্যাব খোলা রাখুন |
| **Smart Poll** | প্রতি ~৩ মিনিটে শিট চেক | আংশিক — scheduler লাগে |
| **External Cron URL** | সার্ভার প্রতি ৫ মিনিটে URL হিট করে | এটাই ক্রনের বিকল্প |

## ২. Action Scheduler কী

WooCommerce-এর ব্যাকগ্রাউন্ড জব সিস্টেম। SheetSync এটা ব্যবহার করে।

**কোথায় দেখবেন:**  
**WooCommerce → Status → Scheduled Actions**

| ট্যাব | মানে |
|-------|------|
| Pending | এখনো run হয়নি |
| Past-due | সময় পেরিয়ে গেছে — **সমস্যা** |
| Failed | error — বিস্তারিত পড়ুন |

**Filter:** Group = `sheetsync`

**সুস্থ:** Pending ০–৫  
**অসুস্থ:** Past-due ৫+ → নিচের [§৬](#৬-background-tasks-overdue-ঠিক-করা) অনুসরণ করুন

---

## ৩. Sync Now (তাৎক্ষণিক)

দ্রুত আপডেট বা প্রথম ইমপোর্টের জন্য সবচেয়ে ভালো।

1. **SheetSync → Connections** → আপনার connection  
2. **Sync** ট্যাব  
3. **Sync Now** (বা Import / Update)  
4. **ব্রাউজার ট্যাব বন্ধ করবেন না** যতক্ষণ প্রোগ্রেস শেষ না হয়  
5. **Products** ও **Sync Logs** চেক করুন

## ৪. Real-time: Smart Poll বনাম Instant

### Smart Poll (Pro)

- Connection → **Enable real-time for this connection** = ON  
- প্রায় **প্রতি ৩ মিনিটে** শিট চেক  
- শিট এডিটের **৩ সেকেন্ড** পর আপডেট হবে না — **৩+ মিনিট** অপেক্ষা করুন অথবা **Sync Now**

### Instant (Apps Script, Pro)

- শিট এডিটের **কয়েক সেকেন্ডের** মধ্যে  
- Connection → Sync → **Optional: Instant sync (Apps Script)** সেটআপ লাগে  

| পদ্ধতি | দেরি |
|--------|------|
| Sync Now | তাৎক্ষণিক (ট্যাব খোলা) |
| Smart Poll | ~৩ মিনিট |
| Cron URL | ৫ মিনিট (সাধারণ সেটআপ) |
| Apps Script | সেকেন্ড |

## ৫. External Cron URL সেটআপ

**সবচেয়ে গুরুত্বপূর্ণ production ফিক্স** যখন “Background tasks overdue” দেখায়।

### ধাপ ১

**SheetSync → Settings** → **Background cron URL**

### ধাপ ২

**Run queue now (test)** ক্লিক করুন — Past-due কমলে ঠিক আছে

### ধাপ ৩

**Cron URL** কপি করুন (টোকেন গোপন রাখুন)

### ধাপ ৪

**Allow background cron URL** চেক → **Save Settings**

### ধাপ ৫ — প্রতি ৫ মিনিটে চালান

**cURL (বেশিরভাগ হোস্ট):**

```bash
*/5 * * * * curl -fsS -m 60 "আপনার-CRON-URL-এখানে" >/dev/null 2>&1
```

**WP-CLI (VPS):**

```bash
*/5 * * * * cd /wordpress/path && wp sheetsync run-queue --seconds=45
```

**EasyCron / cron-job.org:** URL পেস্ট → Every 5 minutes

### ধাপ ৬ — যাচাই

১০ মিনিট পর **Scheduled Actions** → Pending কমেছে কিনা দেখুন

## ৬. “Background tasks overdue” ঠিক করা

হলুদ ওয়ার্নিং মানে ব্যাকগ্রাউন্ড কাজ আটকে।

**দ্রুত ফিক্স:**

1. **WooCommerce → Status → Scheduled Actions**  
2. **Pending** → **Run** (কয়েকবার)  
3. **SheetSync → Settings → Run queue now (test)**  

**স্থায়ী ফিক্স:** [§৫ External Cron URL](#৫-external-cron-url-সেটআপ)

**লোকাল সাইট (wpsync, XAMPP):** WP-Cron প্রায় চলে না — **normal**। Sync Now বা Cron URL ব্রাউজারে খুলুন।

## ৭. সাধারণ সমস্যা

| সমস্যা | সমাধান |
|--------|---------|
| শিট আপডেট করেছি, WC পরিবর্তন হয়নি | ৩ মিনিট অপেক্ষা / Sync Now / Cron সেটআপ |
| প্রোডাক্ট আছে কিন্তু price/SKU নেই | Field Mapping চেক + Sync Now |
| শুধু ১টা variation | [VARIABLE-PRODUCTS-GUIDE.md](VARIABLE-PRODUCTS-GUIDE.md) |
| Row ৩ “no price” warning | Variable parent — **ঠিক আছে**, ignore করুন |
| Cron URL 401 | টোকেন ভুল বা cron disabled |
