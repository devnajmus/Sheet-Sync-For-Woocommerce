## ২. আপলোড ও অ্যাক্টিভেট

### পদ্ধতি A — WordPress Admin (সহজ)

1. পুরনো **SheetSync** ডিঅ্যাক্টিভেট করুন (ডিলিট করবেন না এখনই)
2. **Plugins → Add New → Upload Plugin**
3. নতুন `.zip` আপলোড করুন
4. **Replace current with uploaded** (যদি WP জিজ্ঞেস করে)
5. **Activate**

### পদ্ধতি B — FTP / File Manager

1. `wp-content/plugins/` এ যান
2. পুরনো `sheetsync-for-woocommerce` ফোল্ডার **রিনেম** করুন → `sheetsync-for-woocommerce-old`
3. নতুন zip **আনজিপ** করে একই নামে ফোল্ডার রাখুন
4. Admin → Plugins → SheetSync **Activate**

### আপলোডের পর একবার

- **Settings → Permalinks** → Save (কিছু REST URL-এর জন্য)
- Browser **cache clear** বা Incognito window

## ৩. নতুন ভার্সন ঠিকমতো এসেছে কিনা

| # | কোথায় দেখবেন | কী দেখতে চান |
|---|---------------|--------------|
| 1 | **Plugins** তালিকা | SheetSync active, কোনো fatal error নেই |
| 2 | **SheetSync → Settings** | পেজ লোড হয়; Google JSON আগের মতো আছে |
| 3 | **SheetSync → Settings** → নিচে স্ক্রল | **“Background cron URL”** কার্ড **আছে** (নতুন ফিচার) |

| 5 | **SheetSync → Connections** | আগের connection ঠিক আছে |
| 6 | (ঐচ্ছিক) FTP | `includes/Jobs/class-external-cron.php` ফাইল **আছে** |

যদি **Background cron URL** কার্ড না দেখেন → পুরনো ফাইলই চলছে; আপলোড আবার করুন।

## ৫. লেটেস্ট ফিক্স #১ — সিম্পল প্রোডাক্ট পুরো সিঙ্ক

**সমস্যা ছিল:** শিটে সব ডেটা থাকলেও WC-তে শুধু title, price/SKU/category/image আসছিল না।

### টেস্ট ধাপ

1. WC-তে `DEMO-TSH-001` **ট্র্যাশ** করুন (পুরনো ভাঙা ডেটা মুছতে)
2. Connection → Sync → **Sync Now** (Import বা Update)
3. **ব্রাউজার ট্যাব খোলা রাখুন** যতক্ষণ progress শেষ
4. **Products → All Products** → `Demo Cotton T-Shirt` খুলুন

### ✅ পাস মানে

| ফিল্ড | আছে? |
|-------|------|
| Title | ✅ |
| SKU `DEMO-TSH-001` | ✅ |
| Regular Price | ✅ |
| Categories | ✅ |
| Featured Image | ✅ (URL ম্যাপ থাকলে) |

### ❌ ফেল হলে

- **Sync Logs** দেখুন row 2
- **Field Mapping** — Price, SKU, Categories, Image কলাম ম্যাপ আছে কিনা
- আবার **Sync Now** (৩ সেকেন্ড অপেক্ষা নয় — পুরো sync শেষ হতে দিন)

## ৬. লেটেস্ট ফিক্স #২ — ২টা ভেরিয়েশন সিঙ্ক

**সমস্যা ছিল:** ২টা variation row থাকলেও WC-তে ১টা (যেমন red/s) আসছিল।

### টেস্টের আগে

1. WC-তে `DEMO-HOODIE-01` parent ও সব variation **ট্র্যাশ** করুন
2. শিটে row 4 ও 5 চেক করুন:
   - **Parent SKU** = `DEMO-HOODIE-01`
   - Row 4: Color `red`, Size `s`
   - Row 5: Color `black`, Size `m`
   - প্রতিটি variation-এ **price** আছে
3. **Check sheet** → `2 variations` দেখায় কিনা

### টেস্ট

1. **Sync Now** → ট্যাব খোলা রাখুন
2. Parent `DEMO-HOODIE-01` এডিট করুন
3. **Variations** ট্যাব

### ✅ পাস মানে

| চেক | প্রত্যাশা |
|-----|-----------|
| Variation সংখ্যা | **২টি** (red/s + black/m) |
| প্রতিটির price | শিটের মতো |
| Parent type | Variable product |

### ❌ শুধু ১টা variation থাকলে

1. **SheetSync → Sync Logs** — row 5-এ error আছে কিনা
2. Row 5-এ Parent SKU typo, duplicate Color/Size, বা column U (Variation Attributes) ভুল মান
3. আবার **Sync Now**
4. **[VARIABLE-PRODUCTS-GUIDE.md](VARIABLE-PRODUCTS-GUIDE.md)** § Troubleshooting

## ৭. লেটেস্ট ফিচার — External Cron URL

**কেন:** লোকাল/শেয়ার্ড হোস্টিং-এ Action Scheduler আটকে যায়; এই URL দিয়ে ব্যাকগ্রাউন্ড চালানো যায়।

### ধাপ

1. **SheetSync → Settings** → **Background cron URL**
2. **Run queue now (test)** ক্লিক করুন
3. মেসেজ দেখুন — যেমন: `Past-due tasks: 19 → 3` (সংখ্যা আলাদা হতে পারে)
4. **Allow background cron URL** ✅ → **Save Settings**
5. **Cron URL** কপি করুন

### ব্রাউজারে ম্যানুয়াল টেস্ট

Cron URL ব্রাউজারে পেস্ট করুন (লগইন লাগে না)। JSON বা success মেসেজ আসা উচিত — **401** মানে token ভুল বা feature বন্ধ।

### cURL টেস্ট (টার্মিনাল)

```bash
curl -fsS -m 60 "আপনার_CRON_URL_এখানে"
```

### ✅ পাস মানে

| চেক | প্রত্যাশা |
|-----|-----------|
| Test বাটন | Error না, past-due কমে |
| ৫–১০ মিনিট পর | WooCommerce → Scheduled Actions → Pending কম |
| হলুদ overdue warning | ক্লিয়ার বা কম |

লাইভ সাইটে: প্রতি **৫ মিনিট** cron সেট করুন — **[ব্যাকগ্রাউন্ড-সিঙ্ক-ও-ক্রন-গাইড.md](ব্যাকগ্রাউন্ড-সিঙ্ক-ও-ক্রন-গাইড.md)**

## ৮. Action Scheduler / Overdue ওয়ার্নিং

1. **WooCommerce → Status → Scheduled Actions**
2. Filter: Group = `sheetsync`
3. **Pending** ট্যাবে **Run** (উপরে) — কয়েকবার যতক্ষণ Pending কম না হয়
4. Settings → **Run queue now (test)** আবার

### দেখবেন এসব hook

| Hook | মানে |
|------|------|
| `sheetsync_run_job` | মূল import batch |
| `sheetsync_product_sheet_poll` | Smart Poll |
| `sheetsync_process_media_queue` | ইমেজ ডাউনলোড |
| `sheetsync_job_watchdog` | আটকে যাওয়া job recover |

**Duplicate `sheetsync_job_watchdog` কমেছে কিনা** — নতুন কোড duplicate pending action সরায়; Pending তালিকায় একাধিক একই watchdog থাকলে Run + Cron URL দিয়ে পরিষ্কার হওয়া উচিত।

## ৯. Smart Poll vs Sync Now

| পরীক্ষা | করণীয় | প্রত্যাশা |
|---------|--------|-----------|
| তাৎক্ষণিক | শিটে price বদলান → **Sync Now** | কয়েক সেকেন্ড–মিনিটে WC আপডেট (ট্যাব খোলা) |
| Smart Poll | Real-time ON → শিট এডিট → **৩+ মিনিট** অপেক্ষা | WC আপডেট (cron/scheduler চালু থাকলে) |
| ভুল ধারণা | শিট এডিট → **৩ সেকেন্ড** অপেক্ষা | ❌ আপডেট হবে না — এটা বাগ নয় |

## ১০. Sync Logs দিয়ে যাচাই

**SheetSync → Sync Logs**

| দেখুন | মানে |
|-------|------|
| Row 2 success | Simple OK |
| Row 3 success | Parent OK |
| Row 4 + 5 success | দুই variation OK |
| `skipped` / `unchanged` | শিট ও WC hash মিললে — নতুন ডেটা লাগলে Sync Now |
| Error message | সেই row ঠিক করুন |

লগে row number + SKU মিলিয়ে শিটের সাথে তুলনা করুন।

## ১১. সব ঠিক থাকলে — ফাইনাল চেকলিস্ট

আপলোড সফল ধরে নেওয়ার আগে সব ✅ হওয়া উচিত:

- [ ] Plugin active, কোনো PHP error নেই
- [ ] Settings-এ **Background cron URL** কার্ড দেখা যায়
- [ ] **Run queue now (test)** কাজ করে
- [ ] Check sheet: `4 rows`, `2 variations`
- [ ] `DEMO-TSH-001` — price, SKU, category (ও image যদি ম্যাপ থাকে)
- [ ] `DEMO-HOODIE-01` — **২টি** variation
- [ ] Row 3 “no price” warning বুঝেছেন (parent = normal)
- [ ] Scheduled Actions Pending যুক্তিসঙ্গত (০–৫ ভালো)
- [ ] External Cron URL enable + (লাইভে) ৫ মিনিট cron প্ল্যান করা

## ১২. সমস্যা হলে কী করবেন

| লক্ষণ | প্রথম কাজ |
|-------|-----------|
| পুরনো UI / cron কার্ড নেই | Zip আবার আপলোড; পুরনো ফোল্ডার মুছে নতুন রাখুন |
| Fatal error on activate | PHP 8.0+ আছে কিনা; error log দেখুন |
| Sync শুরু হয় না | Google connection + sheet share |
| Title আছে, বাকি নেই | Sync Now + Field Mapping + Logs |
| ১ variation | Logs row 5 + [VARIABLE-PRODUCTS-GUIDE.md](VARIABLE-PRODUCTS-GUIDE.md) |
| Overdue warning | [ব্যাকগ্রাউন্ড-সিঙ্ক-ও-ক্রন-গাইড.md](ব্যাকগ্রাউন্ড-সিঙ্ক-ও-ক্রন-গাইড.md) |
| Cron URL 401 | Allow cron URL save + সঠিক token |
| Cron URL 429 | প্রতি মিনিট নয় — **৫ মিনিট** interval |

সাপোর্টে পাঠানোর জন্য সংগ্রহ করুন:

1. Sync Logs screenshot (row 4–5)
2. Scheduled Actions screenshot (Pending/Past-due)
3. Check sheet result screenshot
4. Settings → Run queue now (test) এর মেসেজ
